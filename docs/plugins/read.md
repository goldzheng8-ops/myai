# 一、源码阅读（★★★★★ 必会）
| 快捷键                  | 功能                      | 使用频率  |
| -------------------- | ----------------------- | ----- |
| **F12**              | 跳转到定义（Go to Definition） | ⭐⭐⭐⭐⭐ |
| **Alt + ←**          | 返回上一个位置                 | ⭐⭐⭐⭐⭐ |
| **Alt + →**          | 前进到下一个位置                | ⭐⭐⭐⭐  |
| **Ctrl + 鼠标左键**      | 点击进入定义                  | ⭐⭐⭐⭐⭐ |
| **Shift + F12**      | 查找所有引用                  | ⭐⭐⭐⭐⭐ |
| **F2**               | 重命名符号                   | ⭐⭐⭐⭐  |
| **Ctrl + Shift + O** | 当前文件符号（函数/类列表）          | ⭐⭐⭐⭐⭐ |
| **Ctrl + T**         | 全局搜索类、函数、变量             | ⭐⭐⭐⭐⭐ |
